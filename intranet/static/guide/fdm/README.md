# Paquete de guías wiki FDM - Krear 3D

Este paquete contiene 3 guías en formato Markdown para implementar en la web:

1. `guia_errores_comunes_fdm.md`
2. `guia_tips_impresion_3d_fdm.md`
3. `guia_diseno_cad_fdm.md`

También incluye imágenes de referencia extraídas/renderizadas desde los PDFs originales en la carpeta `imagenes/`.

## Uso recomendado
- Usar los Markdown como contenido base para la wiki.
- Mantener la estructura por secciones/acordeones.
- Reemplazar imágenes antiguas por capturas o fotos nuevas cuando sea posible.
- Usar las imágenes incluidas como referencia visual o temporal.

## Notas para sistemas
- El archivo `estructura_guias_wiki_fdm.json` resume los slugs, títulos y CTAs sugeridos.
- Los enlaces de imagen dentro de los Markdown son relativos a la raíz del paquete.

## Actualización incluida

Se agregaron dos guías de software:

- `guia_bambu_studio_fdm.md`: guía principal recomendada para equipos Bambu Lab.
