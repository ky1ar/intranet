# Paquete Wiki LCD / Resina - Krear 3D

Este paquete contiene una propuesta de centro de aprendizaje para equipos de impresión 3D de resina: LCD, MSLA y DLP.

## Enfoque recomendado

Para la web, se recomienda usar **CHITUBOX** como guía principal general para impresión en resina, porque es un laminador ampliamente usado para impresoras LCD/MSLA. Para equipos Anycubic, se puede mantener **Photon Workshop** como recurso alternativo o complementario.

## Archivos incluidos

1. `guia_chitubox_lcd_resina.md`  
   Guía principal para preparar, soportar, laminar y guardar archivos en CHITUBOX.

2. `guia_intro_tecnologia_lcd_resina.md`  
   Explicación simple de la tecnología LCD/MSLA/DLP, materiales, flujo general y conceptos básicos.

3. `guia_seguridad_resina.md`  
   Buenas prácticas de manipulación, protección personal, limpieza y residuos.

4. `guia_postprocesado_wash_cure.md`  
   Flujo recomendado de lavado, secado, retiro de soportes y curado UV.

5. `guia_errores_comunes_lcd_resina.md`  
   Problemas comunes en impresión de resina y cómo diagnosticarlos.

6. `guia_modelos_y_diseno_resina.md`  
   Recomendaciones para elegir modelos, orientar piezas, vaciar modelos, agregar drenajes y preparar piezas para resina.

7. `estructura_guias_wiki_lcd_resina.json`  
   Estructura sugerida para que sistemas cargue los contenidos en la web.

## Carpeta de imágenes

- `imagenes/chitubox/`: capturas extraídas del PDF de introducción a CHITUBOX.
- `imagenes/dlp_resina/`: capturas extraídas del PDF de equipamiento / DLP / postproceso.

Estas imágenes pueden usarse como referencia inicial. Lo ideal es que, antes del lanzamiento, el equipo de contenido reemplace las capturas antiguas por screenshots nuevos del software actual y fotos propias de Krear 3D.

## Comportamiento sugerido en la web

Usar el mismo patrón propuesto para FDM:

- Tarjetas o tabs del Centro de aprendizaje LCD / Resina.
- Al hacer clic en un tema, el contenido se despliega dentro de la misma página.
- El tema anterior se contrae automáticamente.
- Cada guía se muestra como acordeón con índice lateral.
- Mantener opción secundaria: “Descargar PDF”.

## Temas sugeridos para el Centro de aprendizaje LCD / Resina

- Guía rápida de CHITUBOX
- Tecnología LCD / MSLA / DLP
- Seguridad y manipulación de resina
- Postproceso: lavado y curado
- Errores comunes
- Modelos y diseño para resina

## Referencias de contenido

Base interna: PDFs de Krear 3D sobre CHITUBOX, sistema DLP, Photon Workshop, postproceso y errores comunes.

Complemento recomendado: documentación oficial de CHITUBOX para flujos actuales de agregar impresora, cargar modelos, agregar soportes, laminar y guardar archivos.
